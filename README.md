# Costume-Quest-Paint
A Python GUI inspired Paint program that allows painting on a 64x64 grid using images extracted from the Costume Quest PC (Steam) release. Initially made for designing Costume Quest maps easier.

Controls:
- Left Click = Select/Place. (Hold and drag for easier placing)
- Right Click = Delete. (Hold and drag for easier deleting)
- Drag Canvas = Alt + Left Click or Middle Mouse Click then Move.
- Zoom - Scrollwheel or onscreen buttons.

Features I going to try adding:
- Undo / Redo
- Fill Tool
- Object Move & Selection
- Object Search & Favorites
- Grid Toggle
- Exporting
- Quest & NPC Placement
- Recent Files + Auto-save

Limitations/Bugs:
- DLC support will be added.
- Deleting doesn't work currently for Tiles. Just replace the tile instead. This will be fixed.
- Maps are JSON files, these are not images. Opening them will just lead to sadness.
- While fully zoomed out, if you drag the canvas around it can appear laggy. This will likely be fixed or at least improved.
